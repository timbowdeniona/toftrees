# Quickstart

This guide provides the steps to manually test the "Add Grave Inscription" feature.

## Prerequisites

- The application must be running locally.
- You must have access to the Sanity Studio.

## Testing Steps

### 1. Add an Inscription to a Grave in Sanity

1.  Navigate to the Sanity Studio in your browser.
2.  Select a `grave` document to edit.
3.  Find the new "Inscription" field.
4.  Enter some text into the field. Use the bold and italic formatting options.
5.  Publish the changes.

### 2. Verify the Inscription on the Website

1.  Navigate to the grave details page for the grave you just edited.
2.  Verify that the inscription is displayed on the page.
3.  Verify that the bold and italic formatting is rendered correctly.
4.  If the inscription is long, verify that it is displayed in a scrollable pane.

### 3. Test a Grave with No Inscription

1.  Find a grave that does not have an inscription.
2.  Navigate to the grave details page for that grave.
3.  Verify that the inscription section is not displayed.
