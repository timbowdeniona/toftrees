# Data Model: Grave Schema Extension

The `grave` schema will be extended with the following fields:

- **`headstoneCondition`**: `text` - A description of the condition of the headstone.
- **`footstone`**: `boolean` - Indicates whether a footstone is present.
- **`footstoneInscription`**: `text` - The inscription on the footstone.
- **`additionalInformation`**: `text` - Any additional information about the grave.
- **`scenicGraveImage`**: `image` - A scenic image of the grave.
- **`graveImages`**: `array` of `image` - A gallery of images of the grave.
